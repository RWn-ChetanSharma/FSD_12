Loop In JS

The main difference between an entry control loop and an exit control loop is when the test condition is evaluated:

Entry control loop

The test condition is evaluated before the loop body is executed. This type of loop is also known as a pre-checking loop. Examples of entry control loops include the for and while loops.

Exit control loop

The test condition is evaluated after the loop body is executed. This type of loop is also known as a post-checking loop. An example of an exit control loop is the do-while loop.
